Model Architecture

ResNet18, fine-tuned
for binary classification

The model uses a pretrained ResNet18 backbone (ImageNet weights) with its final fully-connected layer replaced by a 2-class output — one for Parasitized, one for Uninfected.

Images are resized to 224×224 and normalized using ImageNet statistics. Training used data augmentation (random flips, rotations) to improve generalization on unseen cell images.

Trained using Adam optimizer with cross-entropy loss over 10 epochs. Best checkpoint saved based on validation accuracy across a 70/15/15 train-val-test split.

Dataset

NIH Cell Images Dataset — 27,558 images, equally balanced between Parasitized and Uninfected cells. Split 70/15/15 for training, validation, and testing.
